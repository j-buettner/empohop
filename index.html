Getting Started

To get started with this project:

  1. Explore the README.md file for an overview of the project structure and features.
  2. Review the Schema Guide to understand the data model.
  3. Examine the Knowledge Graph Data to see how information is structured.
  4. Try the Timeline and Network Graph visualizations.
  5. Use the Data Entry Form to add new events.
